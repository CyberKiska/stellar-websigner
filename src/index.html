Create Signature (.sig)

Flow: choose content -> inspect digest -> sign locally or create unsigned XDR for external wallet -> download.

The app always computes SHA-256 and SHA3-512. Local signing creates a direct SEP-53 content signature; external-wallet signing creates an XDR proof bound to these digests.

Sign With External Wallet (XDR) Generate unsigned XDR, sign externally, paste signed XDR

This app never receives the external wallet secret seed. It generates an unsigned XDR proof request and verifies the returned signed XDR.

Load signer `G...` in Keys before generating XDR, sign on Stellar Public Network in your external wallet, and paste the signed XDR back here.
